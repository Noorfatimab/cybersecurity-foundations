# Day 1 — Binary & Hexadecimal Foundations

## Concepts Learned
- **Bit:** a tiny switch, ON=1 or OFF=0
- **Byte:** 8 bits (8 switches)
- **Binary counting:** each place is a power of 2 (1,2,4,8...)
- **Hexadecimal:** shortcut for 4-bit groups (0–F)

## Examples
- Binary → Decimal: 
  - 1011₂ = 8+0+2+1 = 11₁₀
  - 0101₂ = 4+0+1 = 5₁₀
- Binary → Hex:
  - 10101111 → 1010 1111 → AF
- Hex → Binary:
  - 7F → 01111111

## Practice Exercises
- Convert 11001010 → decimal & hex
- Convert 0x3E → binary & decimal
- Pick a number 1–15, write lamp pattern, binary, hex, decimal

## Why It Matters
Understanding binary and hex is critical for:
- Memory analysis
- Network packets
- Malware reverse engineering
- Certification exams
